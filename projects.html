Connect 4

I created a Human-VS-AI Connect 4 game where players can compete against an AI opponent. The game's solver was written in C++ and compiled to JavaScript-friendly WebAssembly code using Emscripten. The solver uses a negamax algorithm with alpha-beta pruning, and I recently rewrote it to use a bitboard representation of the game board to improve performace(Commit date: Jan 20 2023). The solver is currently able to evaluate 5000 moves per millisecond and play at a depth of 14.
Front-End written in JavaScript. Solver engine written in C++.

Feed-Forward Neural Network

I built a feed-forward neural network using Python and Numpy, without relying on any external ML libraries. This network can be customized with adjustable parameters for depth, input size, hidden layer size, and output layer size. I trained it on the MNIST handwritten digits dataset and the Google QuickDraw! dataset. You can find more information and a detailed explanation of the project at the GitHub link above.
Written in Python using Numpy, Pandas, and Pygame.

ValoTracker

ValoTracker is a player lookup tool for the game Valorant, built using HenrikDev's unofficial Valorant API. It is unique in that it works on every Valorant account, unlike other more popular trackers.

Winding Roads

Winding Roads is a canvas generator that uses a wave-function collapse algorithm to create a visually appealing and logically connected image. The generator takes a set of road sprites and places them onto a grid, ensuring that every road is connected to at least one other road. The end result is a complete image that looks aesthetically pleasing and fits together like a puzzle.
Written in JavaScript
