My Projects

These are some personal projects that I have worked on! They are all works-in-progress, and I will continue to update and improve them as I continue to learn.

Better Leaderboard - Chrome Extension

This is a chrome extension that I made for the game GeoGuessr to fix my biggest complaint with the app. By default, the leaderboard only shows the top 26 players, and the User's only navigation option is a "show more" button that loads the next 26. Better Leaderboard introduces enchanced functionality by giving users the option to filter GeoGuessr leaderboards by username and any custom range. It also allows users to download all of the leaderboard data as a JSON file.
JavaScript(JQuery)

Connect 4

This is a Human-VS-AI Connect 4 game. I wrote the solver in C++ and used Emscripten to compile it to JavaScript-friendly WebAssembly code. In the future, I hope to further optimize the solver by using a bitboard representation and adding a transposition table.
JavaScript with C++ Solver (Emscripten)

Feed-Forward Neural Network

Inspired by Sebastian Lague's Video , I built a feed-forward neural network in Python on top of Numpy arrays, without using any other ML Libraries. The network has an adjustable depth, input size, hidden layer size, and output layer size. I trained it on the MNIST handwritten digits dataset and the Google QuickDraw! dataset. I document the project in-depth at the GitHub link above.
Python(Numpy,Pandas,Pygame)

ValoTracker

ValoTracker is a Valorant player lookup tool. Built on top of HenrikDev's unofficial Valorant API. Unlike more popular trackers, mine works on every Valorant account.
JavaScript

Winding Roads

Winding Roads is a canvas generator that uses a wave-function collapse algorithm to graph a set of road sprites onto a grid, such that every road is connected. The end result is a complete image that looks aestetically pleasing and logically fits together like a puzzle.
JavaScript
